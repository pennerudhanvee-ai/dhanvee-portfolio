 How This Code Works

 HTML Structure

The HTML file (index.html) defines the structure of the webpage:

* **Navbar** → Contains logo and navigation links
* **Hero Section** → Main section with introduction text and avatar
* **Image Tag** → Displays the avatar using a local image file

Example:

```html
<div class="hero">
  <img class="avatar" src="me-1-r.png" />
  <div class="hero-text">...</div>
</div>
```

CSS Styling

The CSS file (`kish.css`) handles the entire design and layout:

1. Layout (Flexbox)

* `.hero` uses `display: flex` to align text and image side by side

2. Background Effects

* Uses dark background + glowing gradients
* `::before` and `::after` create soft light effects

3. Typography

* Uses **Poppins font** from Google Fonts
* Gradient text effect applied to headings

4. Avatar Styling

* The avatar image is styled with:

  * `height: 80vh`
  * `drop-shadow` for glow effect

 5. Buttons & Hover Effects

* CTA button uses gradient + glow
* Hover adds scaling and stronger shadow
